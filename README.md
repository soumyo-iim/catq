# QuizPro

QuizPro is a premium, client-side web application designed to help users practice and solve questions from HTML files locally. 

## Features
- **Hierarchical Topic Organization**: Support for nested folder trees (e.g. `QA/Arithmetic/Ratio/`).
- **Interactive Practice Modes**:
  - **Study Mode**: Real-time feedback, explanations, and instant answer checking.
  - **Exam Mode**: Standard timed testing with performance review.
- **Adaptive Parser Settings**: Inspect and configure custom CSS selectors to handle complex HTML files.
- **Analytics & History**: Track session history and accuracy scores locally.
